(1) What are the advantages of Bootstrap?

ANS -> Speed of development: Bootstrap provides predefined design templates and classes that can save time and effort.
- Responsiveness: Bootstrap enables the creation of websites that are compatible with different devices and screen sizes.
- Consistency: Bootstrap ensures that the design and layout of the website are uniform across browsers and platforms.
- Customizability: Bootstrap can be modified according to the needs and preferences of the project.
- Support: Bootstrap has a large and active community of developers and users who can provide help and feedback.

(2) What is a Bootstrap Container, and how does it work

ANS -> Bootstrap Containers are the most basic layout element in Bootstrap. Bootstrap Containers are very essential and basic building blocks of bootstrap that wrap a page’s content. It’s responsible for setting and aligning content within it according to viewport or given device. Containers are defined within the container class (.container). In other words, we can say that containers are established the width for the layout to give the content. Elements and content are added within the container.
Containers are used for many purposes such as –

- It requires use with the default grid system.
- To establish width for the layout to give web content.
- To provide responsive fixed behavior of any web project.
- To sets the content margin dealing with the responsive behavior of your layout.

(4) What do you know about the Bootstrap Grid System?

ANS ->
- Key features of the Bootstrap Grid System:
(1) Container: The grid system is wrapped in a container. Bootstrap provides two types of containers: .container (fixed-width container) and.

container-fluid (full-width container that spans the entire viewport).

(2) Rows: Rows are used to contain columns, and they should be placed within a container.

(3) Columns: Bootstrap uses a 12-column layout system.

(4) Responsive Classes: Bootstrap provides responsive classes for hiding or showing content based on screen sizes.

(5) Offset Classes: You can create spacing between columns using offset classes.

(6) Nesting: You can nest rows and columns to create more complex layouts.

(5) What is the difference between Bootstrap 4 and Bootstrap 5.

ANS ->
jQuery Dependency:
- Bootstrap 4 relies on jQuery for some of its features, such as the dropdowns and modal components.
- Bootstrap 5 removed the dependency on jQuery, aiming for a more lightweight and modern approach. JavaScript functionality has been rewritten using vanilla JavaScript.

Customization Options:
- In Bootstrap 4, customization and theming were primarily achieved using Sass variables.
- Bootstrap 5 continued with Sass for styling but also introduced a new approach called "CSS Custom Properties" (CSS variables), providing more flexibility and ease of customization for developers.

Grid System Enhancements:
- Bootstrap 4 retained the 12-column grid system but introduced improvements and simplifications. For instance, the gutter width can now be customized using Sass variables.
- Bootstrap 5 also introduced a new responsive grid tier called "xxl" for extra-large screens.

IE 10 and 11 Support:
- Bootstrap 4 supported Internet Explorer 10 and 11.
- Bootstrap 5 dropped support for IE 10 and 11, focusing on modern browsers.

(8) In Bootstrap 4, what is flexbox?

ANS ->
-Flex Container (display: flex;): In Bootstrap 4, the .d-flex class is used to create a flex container. This class is applied to an element to enable the Flexbox behavior.

Flex items go here

-Flex Items: Elements inside a flex container are referred to as flex items. These items can dynamically adjust their size and order based on the Flexbox rules.
-Flex Direction (flex-direction): Bootstrap 4 uses the .flex-row, .flex-column, .flex-row-reverse, and .flex-column-reverse classes to define the direction of the main axis.
Flex items will arrange horizontally
Flex items will arrange vertically

-Justify Content (justify-content): Bootstrap 4 provides classes like .justify-content-start, .justify-content-end, .justify-content-center, and others to align flex items along the main axis.
-Align Items (align-items): Classes such as .align-items-start, .align-items-end, .align-items-center, etc., are used to align flex items along the cross axis.
